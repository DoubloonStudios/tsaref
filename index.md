# The MTHS TSA Reference Guide

This website was created by Mountlake Terrace High School students as a resource for new Technology Student Association (TSA) members to learn the basic skills needed to start competing.

---

## Contributing

We need volunteers to keep this running. Anyone and everyone is welcome! There are many ways you can contribute including:

- Updating information
- Adding new event information
- Finding and fixing bugs or grammatical errors
- Adding images
- Adding cool ideas

To get started, visit the [Github repository](https://github.com/mths-tsa/TSA-Reference) where you can add issues, fix issues, and join the discussion. The [contributing page]({{ "/CONTRIBUTING" | relative_url }}) contains instructions for contributing. You may also [contact the maintainer](mailto:jnac8080@gmail.com) of this repository.

All pages with [U] are unfinished.
